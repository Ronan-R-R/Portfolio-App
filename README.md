
A professional mobile portfolio application built with Flutter to showcase my skills, projects, and professional experience as a Software Engineer.

## Features

- Splash & Personal Brand: High-impact introduction with professional social links.
- Project Showcase: Gallery of past work, featuring a direct integration to my web portfolio.
- Skill Matrix: Visual representation of technical expertise (Flutter, Dart, Web Dev) and soft skills.
- Experience Timeline: A chronological view of my professional history.
- Contact Hub: Functional contact form that integrates with the native email client.

## Tech Stack

- Framework: Flutter (Dart)
- Design: Google Fonts (Inter), Font Awesome Icons, Dark Mode UI
- Dependencies:
  - google_fonts: Typography
  - font_awesome_flutter: Social Icons
  - url_launcher: External link handling

## Getting Started

### Prerequisites
- Flutter SDK installed.
- An IDE (IntelliJ IDEA, Android Studio, or VS Code).
